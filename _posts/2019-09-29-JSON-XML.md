# JSON and XML Examples

JSON: JavaScript Object Notation

{
"firstName": "Tim",
"lastName": "Rayburn",
"address": {
"line1": "5445 Legacy Dr",
"line2": "Suite #100",
"city": "Plano",
"state": "TX",
"zipCode": 75034, 
 } 
}

XML: Extensible Markable Language

<PersonInfor> 
<Name>Tim Rayburn</Name>
<StreetName>5445 Legacy Dr. Suite #100</StreetName>
<City>Plano</City> 
<State>TX</State>
<ZipCode>75034</ZipCode>
</PersonInfor>
